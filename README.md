Design and Evaluation of an End-to-End Breach Lifecycle Simulation in Containerized Environments 📌 Project Overview

This project demonstrates a real-world enterprise breach simulation using Microsoft Azure cloud infrastructure, Docker containers, DVWA (Damn Vulnerable Web Application), and Wazuh SIEM.

The objective of this project is to simulate different cyber attacks in a controlled environment and analyze them using a SIEM platform for monitoring and detection.

🚀 Technologies Used Microsoft Azure Ubuntu 24.04 Docker DVWA Wazuh SIEM Nmap Linux MITRE ATT&CK Framework 🔧 Environment Setup Azure Virtual Machine created Ubuntu 24.04 installed Docker configured DVWA deployed in containerized environment Wazuh SIEM installed and configured 🛡️ Attacks Simulated

Network Scanning Reconnaissance performed using Nmap Open ports identified
SQL Injection Attack Authentication bypass tested Database exposure simulated
Cross Site Scripting (XSS) Script injection attack performed Browser executed malicious payload 📊 Detection & Monitoring

It highlights the importance of monitoring, threat detection, and incident response in cyber security environments.
